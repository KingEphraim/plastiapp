About Us

Plastiqz is a sandbox platform designed for testing and experimenting with payment processing. We provide a safe, reliable environment for businesses and developers to explore payment workflows, test transactions, and fine-tune their systems without risk. With Plastiqz, you can validate your payment solutions with confidence.

Features

Tranzact

This tool is designed for testing payment features within the Sola API and other available integrations. It allows you to simulate transactions, validate responses, and troubleshoot payment processes.

Ewic APL Report

Check the approved ewic list per state.

Settings

The Settings screen lets you configure API keys, toggle between Sandbox and Production modes, manage authentication tokens, and set transaction parameters, ensuring seamless payment testing within the Sola API and integrations.
